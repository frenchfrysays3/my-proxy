# My Proxy
A proxy i made in Node.js

## How to run
1) Install [Node.js](https://nodejs.orgdownload/current) and [Git](https://git-scm.com/install/windows)
2) Clone the repository:

```sh
git clone https://github.com/frenchfrysays3/my-proxy
cd my-proxy
```

3) Install required dependencies: `npm i --omit=dev`
4) Start with `node index`

Go to localhost:3000/?url=https://example.com (or whichever url you wanna go to) and your all set!
